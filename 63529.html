Oracle学习(5)权限

2021年02月24日 295 字 大概 1 分钟

oracle权限

使用scott创建空间权限不足

原因：scott 权限不足

权限的获取

使用：sys / system 账户授权

权限的动作：grant

取消权限的动作：revoko

给予权限

语法：grant 权限 to 用户

给予会话权限：create session

给予创建表权限：any table

一次性给予权限

用来解决 一次性的将相关的权限打包赋予

语法： grant 权限 to 账户;

打包好的权限

connect

​ 是授予最终用户的典型权利，最基本的
​ ALTER SESSION –修改会话
​ CREATE CLUSTER –建立聚簇
​ CREATE DATABASE LINK –建立数据库链接
​ CREATE SEQUENCE –建立序列
​ CREATE SESSION –建立会话
​ CREATE SYNONYM –建立同义词
​ CREATE VIEW –建立视图

resource

​ 是授予开发人员的
​ CREATE CLUSTER –建立聚簇
​ CREATE PROCEDURE –建立过程
​ CREATE SEQUENCE –建立序列
​ CREATE TABLE –建表
​ CREATE TRIGGER –建立触发器
​ CREATE TYPE –建立类型

去除权限

语法：revoke 权限 from 用户

定义

数据定义语言的(ddl)： alter create

数据操纵语言(dml)：select

数据控制语言：grant revoke
