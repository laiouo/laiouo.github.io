Oracle学习(3)命令

2021年02月22日 445 字 大概 1 分钟

oracle命令

访问oracle服务器

自带的SQL Plus

登入：

输入的口令不会显示

普通用户：用户名：scott 口令：(设置的密码)

管理员用户：用户名：system 口令：(设置的密码)

超级管理员用户：用户名：sys 口令：(设置的密码)

一些简单的操作orcle系统的命令

1.查看用户

每条命令后要加结束：“ ; “

show user;

2.切换用户

conn 用户名/密码 as 用户类型(sysdba) ;

sott 和 system 不需要
sys需要填写用户类型

3.修改密码

需要在system账户使用

alter user 用户名 identified by 新的密码;

4.解锁其他用户

alter user 用户名 account unlock;

5.强制登入

(在CMD命令窗口输入)

sqlplus /nolog

链接超级用户：conn /as sysdba;

通过强行登入超级用户可以修改密码

6.查看当前用户下的表

查看表：select * from user_tables;

查看表名：select table_name from user_tables;

如果强制登入链接超级用户显示ERROR:ORA-01031:insufficient privileges

解决方案：

  1. 使用快捷键 “

    Windows+R

    ” 打开运行指令框。在输入框中输入

    compmgmt.msc

    ，回车确定。

  2. 打开了计算机管理，打开本地用户和组 先看 组 中是否有ora_dba；

(若没有本地用户和组请参考：https://jingyan.baidu.com/article/fedf0737eb038c35ac8977c2.html)

\3. 如果组中有ora_dba再看 用户 中找到电脑登录时的当前用户名，一般是第一个

\4. 右键用户名 属性 隶属于看看有没有隶属于 ora_dba

\5. 如果没有，点击 添加，进入“选择组”的选项卡，点击 高级

\6. 先点击 立即查找 搜索结果中选中 ora_dba 最后点击 确定 。

大功告成，一般这样就可以了，快去连接数据库试试吧。
