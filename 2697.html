注意事项

设置的默认值要与列的数据类型一致 例如 数据类型是字符型 ‘112’ 默认值 与 not null / null 位置关系; 正确的位置关系是 默认值 在 not null/null 的前面。

create table student (
name varchar(10) default ‘张三’ null,
birth date,
gender varchar(4),
major varchar(10),
class varchar(10),
tel varchar(11)
)

对表的列进行增删改的操作

alter table student add addr varchar(50);
alter table student modify addr varchar(40);

update student set addr=nvl(null,0);
alter table student modify addr not null;
alter table student modify addr default ‘nanchang’ null;

alter table student drop column addr;
